# galvanotaxis-simulation
This notbook is a toy simulatior for galvanotaxis / electrotaxis intuition building.<br>
y=0 (visualized as the red line on the bottom of the plot) represents zebrafish keratocyte cell surface.

## Current Output
Simulation output is in matplotlib plot format, and saved as animated mp4.

![Watch the MP4 demo](./galvanotaxis_simulation_20260325.gif)
